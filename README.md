# Calculadora de Coisas
Uma calculadora científica com interface gráfica feita em Python através da biblioteca Tkinter. A calculadora conta com funções básicas, como soma, subtração, multiplicação e divisão, assim como funções mais avançadas, como trigonometria, notação científica, fatorial, dentre outras. A aplicação também conta com um conversor de medidas e uma calculadora de datas (em andamento).

## Executando o Programa
Atualmente, a Calculadora de Coisas está funcionando ~~através do executável [thing_calculator.exe]() ou~~ no Windows e no Linux / MacOS, através do interpretador Python. O passo a passo de como rodar a aplicação está descrito logo abaixo:

1. Baixe a pasta compactada do projeto clicando em `<> Code` e depois em `Download ZIP`;

#### No Windows:
2. Instale a versão mais recente do Python através do site [**python.org**](https://www.python.org/downloads/release/python-31210/), de acordo com a arquitetura do seu Windows;
3. Apenas dê um clique duplo no arquivo `run.bat` e ele irá preparar toda a configuração necessária para a execução da Calculadora.

#### No Linux:
2. Abra o terminal na pasta do projeto e execute a seguinte linha de comando: `sudo apt install python3 python3-pip python3-venv python3-tk`;
3. Apenas na primeira vez rodando a aplicação, execute o comando `chmod +x run.sh`;
4. Por fim, execute o comando `./run.sh` e a Calculadora de Coisas irá abrir para você.
