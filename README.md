## Sobre o Aplicativo
Uma calculadora científica com interface gráfica feita em Python através da biblioteca Tkinter. A calculadora conta com funções básicas, como soma, subtração, multiplicação e divisão, assim como funções mais avançadas, como trigonometria, notação científica, fatorial, dentre outras. A aplicação também conta com um conversor de medidas e uma calculadora de datas.

## Executando o Programa
A Calculadora de Coisas possui dois modos de execução:
- Através do arquivo executável (Windows / Linux) disponível em [**Releases**](https://github.com/JGabrielJ/CalculadoraCoisas/releases) ou
- Através do interpretador Python no Windows, Linux e MacOS. O passo a passo de como rodar a aplicação utilizando Python está descrito logo abaixo:

1. Baixe a pasta compactada do projeto clicando em `<> Code → Download ZIP` e depois extraia os arquivos;

#### No Windows:
2. Instale a versão 3.12.10 do Python através do site [**python.org**](https://www.python.org/downloads/release/python-31210/), de acordo com a arquitetura do seu Windows;
3. Apenas dê um clique duplo no arquivo `run_app.bat` e ele irá preparar toda a configuração necessária para a execução da Calculadora.

#### No Linux / MacOS:
2. Abra o terminal na pasta do projeto e execute a seguinte linha de comando: `sudo apt install python3 python3-pip python3-venv python3-tk`;
3. Apenas na primeira vez rodando a aplicação, execute o comando `chmod +x run_app.sh`;
4. Por fim, execute o comando `./run_app.sh` e a Calculadora de Coisas irá abrir para você.
